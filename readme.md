## Client

The client is a react app created using create-react-app. This app relies on the use of a couple dependencies:

- material-ui
- react-router
- redux
- redux-api-middleware



## Server

The server is a very simple Flask server. Its dependencies are:

- Flask
- Flask SQLAlchemy
- SQLlite
